# Vehicle Evaluation System (نظام تقييم المركبات)

## Overview
A web-based vehicle evaluation system with Arabic interface, featuring user authentication and dashboard functionality powered by Supabase.

## Project Structure
- `index.html` - Main login/signup page
- `dashboard.html` - Main dashboard page
- `album.html` - Album/gallery page
- `css/` - Stylesheets
  - `style.css` - Main styles
  - `animations.css` - Animation styles
  - `new-features/` - Additional feature styles
- `js/` - JavaScript modules
  - `supabase-config.js` - Supabase configuration
  - `dashboard-stats.js` - Dashboard statistics
  - `enhanced-features.js` - Enhanced feature implementations
  - `form-validator.js` - Form validation
  - `advanced-filters.js` - Advanced filtering functionality
  - `activity-warehouse.js` - Activity tracking
  - `dark-mode-toggle.js` - Dark mode functionality
  - `enhanced-notifications.js` - Notification system

## Technology Stack
- Static HTML/CSS/JavaScript
- Supabase for backend services (authentication, database)
- Chart.js for data visualization

## Running the Project
The project is served using `npx serve` on port 5000.

## Deployment
Configured for static deployment serving all files from the root directory.
